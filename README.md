# 📖 [Bookstore] <a name="about-project"></a>

> The Bookstore is a website built using React and Redux. It allows user to:
>  - Display a list of books
>  - Add a book
> - and Remove a selected book


## 🛠 Built With <a name="React"></a>

2. HTML.
3. CSS.
4. JAVASCRIPT
5. LINTERS.
6. ES6
